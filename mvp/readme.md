# SmartQueue (AI-Powered MVP)

SmartQueue is a minimal hackathon MVP that replaces physical waiting lines with a
digital queue system. It uses **Google Gemini AI** to intelligently estimate
waiting time based on current queue conditions.

---

## 🚀 What Problem Does It Solve?

Physical queues cause:
- Long waiting times
- Poor crowd management
- No visibility of wait duration

SmartQueue allows users to **join a queue digitally** and instantly know:
- Their position
- People ahead
- AI-predicted waiting time

---

## 💡 MVP Features

- Join a digital queue using a name
- View real-time queue position
- Gemini AI predicts waiting time dynamically
- Simple web-based interface
- End-to-end working prototype

---

## 🛠 Tech Stack

- **Frontend:** HTML, JavaScript
- **Backend:** Node.js, Express
- **AI:** Google Gemini API
